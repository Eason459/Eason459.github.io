前言

我们将会遇到的应用使用几乎都是稀疏图——《算法第四版》

邻接矩阵的处理思路是将顶点和关系分别保存到一个一维数组和一个二维数组中。但是，即使我们保存的是int型数据，一旦数据量达到10万。那么这个数组需要使用的内存空间为：100000 * 100000 * 4Byte = 40GB 所以在这个时候邻接表的优势就很明显了.

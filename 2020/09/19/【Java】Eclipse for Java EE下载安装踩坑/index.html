Eclipse启动出现“An Error has Occurred. See the log file”的解决方案

问题原因

Java EE版本过旧，jdk版本太新！请注重这个问题！！！！旧版本的eclipse是肯定不能兼容新版jdk的！！要么更新eclipse要么安装旧版jdk，个人认为更新eclipse是要更简单的～

问题过程

由于最近开始上c#课程需要用到Java web项目,但是我曾经一直在用的Eclipse是SE版的,虽然SE版可以通过安装相关插件来实现这个功能但是被类似因为插件配置搞的莫名其妙的问题搞怕了，所以还是乖乖下载EE版，下载很简单但是这个致命错误的出现差点给我心态搞崩了，还好我犯的这个愚蠢的问题被我后想明白了：）
